# 🐞 Bug Report - EBAC Shop

Este repositório contém uma atividade prática do curso de Engenharia de Qualidade de Software da EBAC.

## 📌 Objetivo

O objetivo do exercício foi explorar o site de e-commerce de teste da EBAC e identificar problemas de funcionamento (bugs), registrando cada um deles no formato de Bug Report.

## 🔎 O que foi feito

- Navegação pelo site de teste  
- Identificação de erros visuais e de comportamento  
- Registro de 2 bugs encontrados  
- Descrição dos passos para reproduzir cada problema  
- Inclusão de evidências (prints com marcações)

## 🐛 Bugs encontrados

1. Imagens do banner não aparecem na página "Home 1"  
2. Ícone de carregamento (loader) permanece na tela após o carregamento  

## 🛠️ Ambiente

- Produção  
- Navegador: Google Chrome  
- Sistema operacional: Windows 10  
- Acesso via desktop  

## Conteúdo

- Relatório de bugs encontrados no sistema
- Relatório de melhorias sugeridas
